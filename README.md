# Tesloshop

## Backend
1. Clonar repositorio [https://github.com/cgmaldo/react-de-cero-a-experto-2026/tree/main/07-nest-teslo-shop-backend]
2. Clonar archivo .env.template y renombrarlo a .env
3. Configurar variables de entorno
4. npm install
5. Ejecutar el contendor de PostgreSQL con docker compose up -d
6. Lanzar el backend con npm run start:dev
7. La primera vez realizar petición http en Postman o similar para repoblar la base de datos. [http://localhost:3000/api/seed]
8. Será posible ver la descripción del api rest usando la url en el navegador: [http://localhost:3000/api]

## Frontend
1. Ejecutar frontend npm run dev
